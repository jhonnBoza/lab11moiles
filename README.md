# lab11moiles — GLAB-S11

Laboratorio: **Manejo de datos con Core Data y SwiftData**  
Tecsup · Programación en Móviles Avanzado

## Proyecto principal

Abrir en Xcode:

```
lab11jhnn/lab11jhnn.xcodeproj
```

Al ejecutar verás un menú con 3 actividades:

| Actividad | Tecnología |
|-----------|------------|
| 01 Contactos | UIKit + Core Data |
| 02 Alumnos | SwiftUI + SwiftData |
| 03 Productos | SwiftUI + SwiftData (CRUD) |
